## 자기소개

- [x] **학력 및 핵심 역량**
  - 2021.03 ~ 2026.02 경민대학교 컴퓨터소프트웨어 졸업
  - 2026.03 ~ 경민대학교 컴퓨터소프트웨어 (전공심화) 졸업 예정
  - Java, Spring Boot, PostgreSQL을 활용한 백엔드 API 구축 및 데이터베이스 설계 역량 습득
  - 서비스 기획, 시스템 아키텍처 설계, DB 최적화, 최신 AI 기술 연동 등 소프트웨어 개발 생애주기 전반의 실무 경험 함양
<hr>

- [x] **Check-In Club (캡스톤 프로젝트: AI 목표 달성 플랫폼)**
  - 2026.03 ~ 진행 중
  - **서비스 개요:** 사용자가 목표와 세부 행동을 설계하고 매일의 루틴을 기록하면, AI가 행동 패턴을 분석해 더 나은 방향을 제안하는 플랫폼. 레이싱 세계관(트랙, 자동차 등)을 접목하여 사용자의 지속적인 목표 달성 동기를 유발하도록 기획.
  - Spring Boot 기반 백엔드 API 서버 설계 및 카카오 소셜 로그인(OAuth 2.0) 사용자 인증 로직 구현
  - 데이터 무결성 및 응답 속도 향상을 위한 PostgreSQL 3정규형 DB 설계, 복합 인덱스 적용 및 서브쿼리(EXISTS) 튜닝
  - Ollama 프레임워크 기반의 오픈소스 LLM(Qwen 2.5)을 연동하여 사용자 행동 패턴을 분석하는 AI 비즈니스 로직 개발
  - 무거운 AI 연산으로 인한 서버 멈춤(Blocking) 현상을 메시지 큐 기반의 비동기 아키텍처로 완벽히 분리하여 병목 해소
  - 단순 사진 업로드 인증의 꼼수를 막기 위해 OCR, Vision AI 및 GPS 위치 정보를 결합한 엄격한 다중 검증 시스템 기획 및 적용
<hr>

- [x] **Scan Cook (스마트 냉장고 앱 프로젝트)**
  - 2025.03 ~ 2026.02
  - React Native를 활용한 스마트 냉장고 앱 프론트엔드 UI 및 클라이언트 비즈니스 로직 구현
  - 식재료 유통기한 관리 및 맞춤형 레시피 추천 데이터를 화면에 렌더링하기 위한 백엔드 API 연동
  - Google ML Kit의 Barcode Scanning 기술을 도입하여 카메라 영상 내 상품 바코드 데이터를 실시간으로 빠르고 정확하게 추출하는 로직 구현
  - 바코드 인식률 저하 문제를 해결하기 위해 이미지 인식 영역(ROI)을 화면 중앙으로 좁히고 조도 노출 값을 튜닝하여 인식률 20% 이상 향상
  - 실시간 스캔 시 발생하는 과도한 중복 API 요청을 막기 위해 프론트엔드 단에 디바운싱(Debouncing) 로직을 적용하여 백엔드 서버 부하 사전 방어
